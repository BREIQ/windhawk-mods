## üöÄ Featured Mods

### üßπ [Clean Desktop on Idle](taskbar-idle-clean-desktop/)
Automatically tidies your workspace by hiding the taskbar and minimizing windows after a period of user inactivity.

* **Smart Detection:** It **will not** interrupt you while watching movies, streaming, or playing games.
* **Instant Restore:** Your taskbar and workspace return to normal as soon as you move the mouse or touch the keyboard.
* **Privacy:** Perfect for clearing your screen automatically when you step away from your PC.

## üõ†Ô∏è How to use

1.  **Install Windhawk:** Download and install it from the [official website](https://windhawk.net/).
2.  **Find the Mod:**
    * **Option A:** Search for the mod name directly in the **Windhawk Official Catalog**.
    * **Option B:** Copy the source code from the `.cpp` file in this repository and paste it into the "New Mod" editor within Windhawk.
3.  **Configure:** Adjust the idle timer and other settings in the "Settings" tab of the mod.
